Exemple de configuration de vos applications CARL Cliquez ici.

CARL Source Admin

Configuration d'une instance CARL Source Admin: 
Mail : Serveur SMTP	| localhost ( addresse du serveur CARL MAil dev)
Mail : Port SMTP	| {{ config.smtpPort }} 
Mail : Adresse de l'expéditeur	| CARL@TEST-CLIENT.com ( email non existant de votre choix)

Jboss 4.2.X

Configuration du fichier mail-service.xml de l'instance jboss: 

<!-- Change to the SMTP gateway server -->
<property name="mail.smtp.host" value="localhost"/>
 
<!-- The mail server port  -->
<property name="mail.smtp.port" value="25"/> -->
      
<!-- Change to the address mail will be from  -->
<property name="mail.from" value="noreply@domain.com"/>
